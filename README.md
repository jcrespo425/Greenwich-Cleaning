# Greenwich Cleaning

Polished multi-page website for `greenwichcleaning.com`.

## Pages

- `index.html`
- `services.html`
- `about.html`
- `service-area.html`
- `contact.html`

## Local development

```bash
npm install
npm run dev
```

Or double-click `start-dev.bat`.

## Production build

```bash
npm run build
```

The production output is generated in `dist/`.

Or double-click `start-preview.bat` to build and preview.

## What changed in the polished pass

- Rebuilt the site around the real brand name: `Greenwich Cleaning`
- Added real photography to improve credibility
- Expanded page content depth and local market positioning
- Added canonical tags, Open Graph tags, and JSON-LD schema
- Improved the service and quote flow structure

## Before launch

- Replace `(203) 203-0000` if you plan to use a different business line
- Confirm `hello@greenwichcleaning.com` is the launch inbox
- Replace stock photography with real business or family-approved photos if available
- If you want full form handling, replace the current `mailto:` flow with Formspree, HubSpot, Netlify Forms, or a custom endpoint
- Add real testimonials once you have permission to publish them

## Stock photo placeholders used

- Unsplash interior and home photography was downloaded into `public/images/` for this build
- These should ideally be replaced with real brand photography for the strongest final launch
